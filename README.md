Dice Game – JavaScript, HTML5 & CSS3
Dice Game is a simple and fun game where two players roll dice at the same time. The player with the higher number wins.

Features
Designed for two players.
Dice are displayed in 2D.
The player with the higher dice wins.
Built with HTML5, CSS3, and pure JavaScript.
How to Use
Clone the repository or download the ZIP file:
git clone https://github.com/yourusername/dice-game.git

Open index.html in your browser.
Click the “Roll Dice” button to start the game.
After the dice roll, the winner will be displayed on the screen.
